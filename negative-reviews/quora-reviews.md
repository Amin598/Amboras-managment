## Patterns (for NOTES.md)

### Validated existing patterns:
- **App overload / dependency** — confirmed across multiple Quora threads
- **High fees / hidden costs** — transaction fees, refund fee retention, paywalled features
- **Support quality** — non-existent, gated behind expensive plans
- **SEO limitations** — structural problem (URLs, sitemaps, duplicate content)
- **Platform lock-in** — migration difficulty, data portability concerns
- **Fund holds / account freezing** — Shopify holding money hostage
- **Store shutdowns** — arbitrary, no explanation, data loss

### New/reinforced patterns from Quora:
- **No marketplace = no traffic** — Shopify charges fees but provides zero organic discovery (unlike Etsy, Amazon)
- **Predatory expert ecosystem** — "experts" are scammers, pay-then-block pattern
- **Product variant limits** — 100 variants / 3 options is a hard ceiling frustrating merchants with complex catalogs
- **Deceptive pricing** — $30/mo advertised but $300/mo needed for basic tax/customs features
- **Outdated theme technology** — themes not using modern tech, hard to modernize
- **Platform designed for Shopify's profit, not merchant success** — "only people that make money from Shopify dropshipping is Shopify"
- **False expectation setting** — merchants believe building a store = getting sales, platform does nothing to correct this
- **Analytics hit a wall** — dashboard useless past basic sales totals
- **Refund fee retention** — Shopify keeps processing fees even on full refunds
- **International transaction penalty** — 4-6% per international transaction
